# ECCB2026 Workshop Template Repository

Template repository for a full-day workshop on biological networks, multi-omics integration, and LLM-agent workflows for biomedical knowledge graphs.

## Workshop Agenda

### Session 1 - Introduction to Biological Networks and Knowledge Graphs
- 09:10 - 09:40: What is a Network and a Knowledge Graph?
- 09:40 - 10:10: Knowledge Graph Development using NetworkX
- 10:10 - 10:45: Practical Session 1 - Creating a Phenotype Knowledge Graph from ICD-10 Codes
- 10:45 - 11:00: Coffee Break

### Session 2 - Creating Multi-Omics Profiles
- 11:00 - 11:30: Linear Methods for Multi-Omic Integration
- 11:30 - 12:00: Deep Learning Approaches for Multi-Omic Integration
- 12:00 - 13:00: Practical Session 2 - Multi-Omics Interpretation with MOFA
- 13:00 - 14:00: Lunch

### Session 3 - Building Agentic LLM Workflows for Biomedical Knowledge Graphs
- 14:00 - 14:30: What is an LLM agent?
- 14:30 - 15:00: Tool Use and Model Context Protocol (MCP)
- 14:30 - 16:00: Practical Session 3 - Querying LLM agents with Molecular Profiles
- 16:00 - 16:15: Coffee Break

### Session 4 - Knowledge Graph Query using Multi-Omics and LLMs & Mini-Challenge
- 16:15 - 16:45: How to Query a Knowledge Graph using Multi-Omic Profiles via LLMs
- 16:45 - 18:00: Practical Session 4 - Mini-Challenge
- 17:50 - 18:00: Closing Remarks

## How to Use This Template

1. Copy this repository as your workshop starter repo.
2. Add slides, notebooks, datasets, and exercises inside each session folder.
3. Use each practical folder's checklist to prepare hands-on material.
4. Customize timing and instructions based on your audience.

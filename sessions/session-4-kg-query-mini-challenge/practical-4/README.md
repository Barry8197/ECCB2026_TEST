# Practical 4 Template: Mini-Challenge

## Objective
Apply workshop concepts to answer a biomedical question using multi-omics signals, knowledge graph context, and LLM assistance.

## Challenge Prompt Template
1. Define the biomedical question
2. Specify available omics and graph inputs
3. Design a query and agent workflow
4. Summarize findings and limitations

## Submission Checklist
- [ ] Problem statement
- [ ] Data and graph resources used
- [ ] Query/agent workflow description
- [ ] Final result summary
- [ ] Limitations and next steps
